# Service Page SEO

## Typy stron usługowych

Wg `inc/service-pages.php`:
- `pracownia-zlotnicza` - pracownia złotnicza
- `bizuteria-na-zamowienie` - biżuteria na zamówienie
- `naprawa-bizuterii` - naprawa biżuterii

## SEO dla usług

### Title template
```
[Usługa] | [Marka] - Kraków
```

Przykład:
```
Pracownia złotnicza Kraków | BigDIAMOND - Indywidualne projekty
```

### Meta description
```
[Usługa] w Krakowie. [Krótki opis]. Zadzwoń i zamów projekt.
```

### Content struktura

1. Hero z nazwą usługi
2. Opis usługi
3. Proces (jak to działa)
4. Ceny / wycena
5. FAQ
6. CTA (kontakt, zamówienie)

## Local signals

- Adres w treści
- Mapa Google
- Telefon
- Godziny otwarcia
- Local schema (LocalBusiness/Service)
